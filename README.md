# 1701 Industries Website

Static HTML/CSS website with English, Portuguese, and Japanese pages.

## Structure
- `/index.html` English homepage
- `/about.html`
- `/services.html`
- `/why-1701.html`
- `/contact.html`
- `/pt/` Portuguese pages
- `/ja/` Japanese pages
- `/assets/styles.css`
- `/assets/script.js`

## Publish on GitHub Pages
1. Create a new GitHub repository.
2. Upload all files from this folder.
3. Go to **Settings** -> **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and the `/root` folder.
6. Save.

Your site will publish on the GitHub Pages URL for that repository.
