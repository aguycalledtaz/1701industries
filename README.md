# 1701 Industries Website

Static multilingual website for 1701 Industries.

## Included
- `index.html`
- `styles.css`
- `script.js`
- `vercel.json`

## Languages
- English
- Portuguese
- Japanese

## Local preview
Open `index.html` in a browser, or run a simple local server.

### Python
```bash
python3 -m http.server 8000
```
Then open `http://localhost:8000`

## GitHub Pages deployment
1. Create a new GitHub repository.
2. Upload all files from this folder to the repository root.
3. Commit and push.
4. In GitHub: `Settings` → `Pages`.
5. Under `Build and deployment`, choose `Deploy from a branch`.
6. Select the `main` branch and `/root` folder.
7. Save.
8. Your site will publish on the GitHub Pages URL.

## Vercel deployment
1. Create a new project in Vercel.
2. Import the GitHub repository.
3. Keep the default settings.
4. Deploy.
5. Point your domain to Vercel when ready.

## Notes
- Update the email address in `index.html` if needed.
- Replace text content directly in `script.js` if you want to revise copy in all languages.
- Typography is intentionally restrained for a minimal corporate look.
